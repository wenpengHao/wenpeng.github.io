如何使用hugo搭建个人博客

Posted on December 1, 2018
1. mac 下安装hugo
#安装命令
brew install hugo
#查看版本
brew version
Hugo Static Site Generator v0.52 darwin/amd64 BuildDate: 2018-11-28T14:06:34Z
  • hugo documentation : https://gohugo.io/getting-started/quick-start/
  • hugo github: https://github.com/gohugoio/hugo/releases
2. 创建博客
hugo new site blog.demo.com

## 
hugo new posts/new.md
3. 下载主题
运行博客
#开发状态下
hugo server --theme=beautifulhugo  --buildDrafts  -d dev
部署博客到个人网站
#编译
注意事项
1. 去掉文章的草稿编译状态，默认 draft:false 
draft:false -> dragt:true
2. 记得去看官方文档
